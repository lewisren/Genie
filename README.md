How-To-Run Yolo-Robot
---------------------
### Setup
Get Brew

Get Python
	$ brew install python

Get Pip
	$ easy_install pip

Get NumPy
	$ sudo pip install numpy

Get Fortran
	$ brew install gfortran

Get SciPy
	$ sudo pip install scipy

Get Matplotlib
	$ brew install pkg-config
	$ sudo pip install matplotlib

Get SciKit Learn!
	$ sudo pip install -U scikit-learn
